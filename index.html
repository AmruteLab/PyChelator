Example free Ca 2+ calculation

Solution A having pH 7.4, ionic equivalent 100 mM at 4°C, and containing:

  • 5 mM EGTA (98% purity)
  • 1 mM CaCl2
  • 2 mM MgATP (97% purity)
  • 2 mM MgCl2

Steps for free Ca 2+ calculation in PyChelator:

  1. In the Constants dropdown, keep the default NIST database constants, or choose another preferred source.
  2. Select mM as Input unit.
  3. Write the Temperature as 4°C.
  4. Write the pH as 7.4.
  5. Write the Ionic (Ie) equivalent as 100 mM. If you do not know the ionic equivalent, you can calculate by using the Calculate Ie.
  6. Select the parameters to Show in report, in this case we will keep it at defaults.
  7. Using the top panel, write the Chelator concentration of ATP to 2 mM and EGTA to 5 mM. Also, write the Total metals given to the solution; Ca²⁺ as 2 mM and Mg²⁺ as 4 mM?
  8. Click Calculate. You can copy the Free (M) concentration for Ca²⁺ (1.678e-8 M) or the -log10[free] (pCa of 7.7752). Alternatively, click Download to have a local copy of the results.

Example manual entry of stability constants

Let’s use the constants for Zinc and Calcium buffered using EDTA and EGTA, from Neumaier et al. 2017.

Steps for manual entry of constants in PyChelator:

  1. Under the Constants dropdown, find Write custom constants
  2. Write the Ligands (L1, L2) comma-separated as: EDTA, EGTA.
  3. Write the Metals (M1, M2) comma-separated as: Zn, Ca.
  4. Write the temperature that the constants were determined (Et) in °C: 20.
  5. Write the ionic equivalent that the constants were determined (Ei) in mM: 100.
  6. Write the protonation constant H1 (logK) for L1 and L2: 10.3, 9.5
  7. Write the ΔH1 enthalpy (kcal/mol) for H1 for L1 and L2: -5.6, -5.9
  8. Repeat steps 6 and 7 for all protonation constants.
  9. Write the stability constant ML (logK) for M1-L1,M1-L2,M2-L1,M2-L2: 16.6, 12.7, 10.7, 10.9
  10. Write the ΔML enthalpy (kcal/mol) for M1-L1,M1-L2,M2-L1,M2-L2: -4.7, -4.3, -6.1, -8.4
  11. Repeat Steps 9 and 10 for MHL and ΔMHL, respectively.
  12. Download the JSON file. You can use this file the next time using the command Upload constants under Constants dropdown.
